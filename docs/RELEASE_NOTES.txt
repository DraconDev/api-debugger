✨ NEW IN THIS VERSION

🔍 REQUEST CAPTURE
• Automatically captures ALL HTTP requests
• View headers, body, status, timing
• Filter by domain, method, status
• Works on every website

🔨 REQUEST BUILDER
• REST, WebSocket, GraphQL, SSE, Socket.IO
• Headers, params, body, auth
• Pre-request scripts (Postman-compatible)
• Collections, environments, variables

📥 IMPORT / 📤 EXPORT
• Import from: Postman, Insomnia, OpenAPI, HAR, cURL
• Export to: Postman, OpenAPI, cURL
• No lock-in

🤖 AI INTEGRATION
• Bring YOUR OWN API key
• Uses OpenRouter (300+ models)
• Default model is FREE
• Your keys NEVER touch our servers

🔐 PRIVACY FIRST
• NO account required
• NO telemetry, NO tracking
• Data stored LOCALLY in your browser
• GitHub sync uses YOUR token only

🌐 BROWSER SUPPORT
• Chrome (Manifest V3) ✅
• Firefox (Manifest V2) ✅
• Edge (Chromium) ✅

📋 WHAT'S THE STORY?

I was debugging an API at 11pm.
Postman wanted to verify my email. Again.

So I built a Chrome extension that captures
every request my browser makes.

No account. No login. No "sorry, free tier
requires email verification."
