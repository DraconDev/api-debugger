## Chrome Web Store

**Title**: API Debugger - Capture & Debug HTTP Requests

**Short Description** (95 chars):
Inspect every HTTP request. Capture, replay, and debug APIs. No account needed.

**Description** (full):

```
Inspect every HTTP request your browser makes. Capture, replay, build, and debug APIs with full privacy.

WHAT YOU GET:
• Auto-capture all browser HTTP traffic
• Build requests: REST, WebSocket, GraphQL, SSE, Socket.IO
• Organize with collections and environments
• Pre/post scripts (Postman-compatible syntax)
• AI-powered request analysis (bring your own key)
• Import from Postman, Insomnia, OpenAPI, HAR, cURL
• Export to Postman, OpenAPI, cURL

AI INTEGRATION:
Uses your own OpenRouter API key - we never see your data. Default model is free.

PRIVACY FIRST:
• No account required
• All data stored locally
• No telemetry, no tracking
• GitHub sync uses YOUR token - we never see it

Perfect for developers debugging APIs, testing integrations, or inspecting traffic.

Need help? Have questions? Visit YOURSITE.COM for support and other tools.
```

## Privacy Policy

For Chrome Web Store, you need a privacy policy URL. Options:

1. **GitHub Pages** (free)
   - Create a repo: `username.github.io`
   - Add simple privacy page

2. **Short privacy policy** (can embed in store):

   ```
   API Debugger Privacy Policy

   This extension stores data locally on your device:
   - Collections, requests, and settings: chrome.storage.sync
   - Request history: chrome.storage.local

   We do not:
   - Collect any personal data
   - Send telemetry or analytics
   - Access your API requests or responses
   - Share data with third parties

   AI features use your own API keys (OpenRouter) - we never see them.

   GitHub sync uses your personal access token stored locally.

   For questions: open an issue on our GitHub repository.
   ```
